这是介于之前发布的《 使用 svg 后如何改变颜色》 的第四种解决方案。

由于方案三使用的是 CSS 解决，总的来说在部分场景下会不太方便，并且 transform: translate(-1000px); 的值始终不好控制，以至于在某些场景下会导致图片“消失”，就比较诡异。

方案四

避免使用 <img> 标签，而是直接使用脚本自动化解析 svg 文件，通过规律生成一个新的 JSX 组件子集，以此来解决问题。
