源码中以 match 函数来对参数进行识别，以条件选项最多的 mapDispatchToProps 为例子。

  1. whenMapDispatchToPropsIsFunction 传入识别为函数
  2. whenMapDispatchToPropsIsMissing 未传入
  3. whenMapDispatchToPropsIsObject 传入识别为对象
